一、AI 工具使用及生成限制规范

允许使用范围：仅可使用 AI 直接生成人声、或将 AI 生成人声作为作品采样（Sample）使用。除此之外的 AI 应用均属违规。

严禁使用行为：绝对禁止使用 AI 全自动生成整首歌曲，包括但不限于 AI 作曲、AI 编曲、AI 自动配器、AI 自动混音、AI 生成伴奏等核心创作与制作环节。

禁止变相重构：禁止利用 AI 对作品进行一键重构、自动编曲、自动补齐结构、自动修曲等变相生成行为。

声线合法性要求：AI 生成人声须来源合法，严禁克隆、模仿任何第三方歌手、网红或他人声线。AI 人声仅限作为音色或采样使用，不得大量拼接拼凑并作为作品核心结构。

审核裁决权：厂牌对疑似全套 AI 生成、变相重构的作品拥有独立且唯一的鉴别裁决权。创作者有义务在厂牌提出质疑时，配合提供分轨工程录屏或分轨音频（Stem）以自证清白。

二、作品制作质量与音频技术审核标准

所有投稿作品需满足制作精良、专业度合格的基础要求，凡出现以下任一问题，直接判定审核不通过：

  1. 音乐基础素养：存在明显跑调、节奏混乱、旋律无逻辑、和声搭配错误等基础音乐问题。Demo、半成品、无完整编曲、无合理结构的作品，一律不予通过。
  2. 技术参数与音质缺陷：存在明显底噪、喷麦、混音失衡、电平爆顶、音质劣化等问题。
  3. 风格接纳范围：本厂牌面向全品类流行电子风格及融合电子曲风开放投稿，包括但不限于 House、Phonk、Future Bass、Brazilian Funk、Synthwave、Cyberpunk、Electronic Pop（流行电音）等。曲风混杂、主题表达极度混乱者不予收录。
  4. 速成拼凑与洗稿：仅靠模板拼接、工具拼凑完成的作品，或旋律高度相似、照搬热门曲目、涉嫌洗稿的，一律按抄袭处理。
  5. 歌词内容合规：歌词内容低俗、涉黄、涉毒、涉政、具有侮辱性或极端负能量导向的，直接不予通过。

三、全球版权合规与排他性分发硬性要求

版权合规为作品审核一票否决项，所有作品必须满足以下版权要求，违者从严处理：

  1. 排他性数字分发权：作品审核通过后，投稿方须授权本厂牌在合作期限内，享有该作品在全球范围内的排他性数字音频分发与流媒体发行权（含邻接权及获酬权），以及非独占的宣传推广权。创作者须保证在合作期内，不得自行或授权第三方重复分发、上架该音频，严禁一稿多投。
  2. 素材全球商业授权：由于全球发行涉及 YouTube Content ID 及各海外平台版权自动抓取系统，投稿作品中所使用的所有伴奏（Beat）、采样（Sample）、音效、人声素材等，均需具备完整合法的全球范围商业使用授权文件。严禁使用无版权、盗版、侵权素材或"仅限非商业"素材。
  3. 改编与翻唱授权：翻唱、改编、Remix 作品须提供原曲版权方正式书面授权，无授权不予审核。标注"免费"但限制商业发行、厂牌上架或收益分成的素材，均不得使用。
  4. 法律与经济责任承担：若作品涉及版权纠纷、侵权投诉等法律问题（包括但不限于海外平台下架、封线或扣除收益），全部法律责任与经济责任由创作者自行承担，本厂牌不承担任何连带责任。

⚠️ 全球收入结算清晰指引（核心约束项）：

  1. 结算周期与发放频率：本厂牌实行季度结算制（每三个月发放一次收入）。
  2. 统一汇总与发放逻辑：基于本作品通过发行平台（秀动）进行的全球分发，其在国内平台（网易云音乐、QQ 音乐、酷狗等）及海外平台（Spotify、Apple Music、YouTube 等）所产生的所有数字流媒体收益、版权收益及衍生版税，不会自动直接打入创作者个人的音乐人后台。上述全网收益将统一汇总至发行方（SiriusRecords）的秀动厂牌主理人后台，由厂牌方进行代收并结算。
  3. 运营费用扣除与纯利分成：每季度的收入在发放前，将首先扣除平台收取的必要运营费用（如流媒体平台基础扣款、跨境代发手续费或平台硬性税费等）。扣除运营费用后的净利润，严格按照 发行方 20%、投稿方 80% 的分成比例拆分为两份进行境内最终发放。
  4. 结算延迟悉知：鉴于全球流媒体平台（尤其是海外渠道）出账存在不可控的周期延迟（通常在海外出账后存在 2–3 个月的结算延迟）及多币种汇率实时波动，所有季度结算数据均严格依据发行平台实际出账的正式账单及银行实际到账数据为准。

五、专辑活动投稿格式与流程指引

所有作品统一投递至官方邮箱：2972444840@qq.com

邮件主题格式：【XX专辑投稿】+ 作品名 + 演唱者

⚠️ 强制打包要求：投稿方必须将所有相关文件整合并统一打包为唯一的 .zip 或 .rar 压缩包 作为邮件附件发送，不接受任何散落的零碎附件或未打包文件。

邮件正文或附件文档须完整填写以下信息：

  1. 歌名：
  2. 演唱者：
  3. 作曲者：
  4. 编曲者：
  5. 投稿类型：原创 / Remix

音频技术文件（压缩包内必需）：

  1. 正式分发用母带：必须为无损 WAV 格式，16-bit / 44.1kHz 或更高。
  2. 快速试听文件：必须包含全曲 MP3 格式音频，用于厂牌初审。

版权与原创证明文件（压缩包内必需，一票否决项）：

  1. 原创作品防洗稿 / 侵权倒查核验：
    • 必须提交该作品的 DAW 宿主工程源文件（如 .flp / .als / .cpr 等）。
    • 必须提交工程中关联的原始采样文件 / 素材包（Samples / Loops），确保工程文件能在厂牌设备上完整加载且不丢失采样路径。
    • 【高压防线】：工程源文件内所有的音频切片（Slices）和 Loop 采样绝对禁止提前进行"冻结（Freeze）"或"渲染合并（Bounce / Flatten）"等不可逆处理。所有采样必须保留原始状态。厂牌技术审核团队将通过直接盘查工程、溯源原始采样包，倒查是否存在盗用未授权版权或抄袭等洗歌行为。拒绝提供或提供空工程、隐藏音轨者，直接不予通过。
  2. 附原创授权许可、购买的商业 License 文本（如使用了第三方付费素材包）。
  3. 改编作品：附原作品版权方正式书面改编、Remix 授权许可书面文件。
  4. 分成协商安排明细：明确写明投稿方内部（词、曲、编、唱等）的具体百分比分配方案。

六、违规处理与附则
